Preliminaries #

Install Hanfor #

Clone the repository:

$ git clone https://github.com/ultimate-pa/hanfor.git -b master --single-branch /your/hanfor/destination

Hanfor requires Python and is only tested with Python 3.6.x. You can check if you have python already installed from the command line:

$ python -- version
Python 3.6.2

We recommend using a virtual environment. Create a new virtual environment with: 

$ virtualenv hanfor_python

And activate it by sourcing:

$ source hanfor_python/bin/activate

Now the python dependencies needed to be installed into the virtual environment. Inside the repository run:

$ pip install -r requirements.txt
